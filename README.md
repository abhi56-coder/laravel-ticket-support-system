# Laravel Helpdesk System

A Laravel-based helpdesk project for user requests, agent assignment, priority management, replies, attachments, and progress tracking.

## Project Goal

This project demonstrates Laravel development skills through a real helpdesk workflow. It is useful for PHP, Laravel, product support, implementation, and technical consultant roles.

## Tech Stack

- Laravel 10/11
- PHP 8+
- MySQL
- Blade / Bootstrap
- Git + GitHub

## Planned Features

- User login
- Request creation
- Request assignment
- Priority levels
- Status tracking
- Comments and replies
- File attachment planning
- Agent dashboard
- Admin dashboard
- Search and filters

## Main Modules

1. Authentication
2. Requests
3. Categories
4. Comments
5. Attachments
6. Users and Agents
7. Dashboard
8. Reports

## Database Tables

- users
- requests
- request_categories
- request_comments
- request_attachments
- activity_logs

Built a Laravel-based Helpdesk System with request creation, assignment workflow, priority/status tracking, comments, attachment planning, MySQL schema design, and dashboard documentation.
